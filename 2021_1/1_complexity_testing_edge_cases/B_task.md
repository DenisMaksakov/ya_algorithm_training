# B. Треугольник

|                     |                                  |
| ------------------- | -------------------------------- |
| Ограничение времени | 1 секунда                        |
| Ограничение памяти  | 64Mb                             |
| Ввод                | стандартный ввод или input.txt   |
| Вывод               | стандартный вывод или output.txt |

Даны три натуральных числа. Возможно ли построить треугольник с такими сторонами. Если это возможно, выведите строку `YES`, иначе выведите строку `NO`.

Треугольник — это три точки, не лежащие на одной прямой.

## Формат ввода

Вводятся три натуральных числа.

## Формат вывода

Выведите ответ на задачу.

### Пример 1
Ввод
```
3
4
5
```
Вывод
```
YES
```

### Пример 2
Ввод
```
3
5
4
```
Вывод
```
YES
```

### Пример 3
Ввод
```
4
5
3
```
Вывод
```
YES
```

## Решение

**Суть:** Проверка выполнения неравенства треугольника для трех сторон.

**Математическая основа:**
Треугольник существует тогда и только тогда, когда для его сторон a, b, c выполняется:
```
a + b > c
a + c > b  
b + c > a
```

**Логика алгоритма:**
1. Получить три натуральных числа (стороны)
2. Проверить все три условия неравенства треугольника
3. Если все условия выполняются → `YES`, иначе → `NO`

**Сложность:**
- **Время:** O(1) - константное время, только три сравнения
- **Память:** O(1) - фиксированное количество переменных

**Ключевые особенности:**
- Детерминированный результат за один шаг
- Не зависит от порядка ввода сторон
- Простая условная логика без циклов
- Обрабатывает все возможные комбинации сторон
